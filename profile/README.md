# Опенсорсеры
Русскоязычное сообщество опенсорс-разработчиков.

---

## Цель сообщества
Обмен опытом и совместная работа между разработчиками.

---

## Как присоединиться
1. Подписывайтесь на репозиторий
2. Участвуйте в обсуждениях
3. Вносите свой вклад в проекты
4. Делитесь своими проектами и идеями
